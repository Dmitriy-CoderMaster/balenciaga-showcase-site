# BALENCIAGA — Concept Store (HTML/CSS/JS)

Самодостаточный одностраничный сайт-каталог. Никаких сборок и зависимостей.

## Как запустить локально
Просто открой `index.html` двойным кликом. Всё работает.

## Как залить на GitHub и поделиться ссылкой
1. Создай новый репозиторий на GitHub.
2. Скопируй файлы из этой папки (`index.html`, `styles.css`, `app.js`) в **корень** репозитория.
3. В репозитории: **Settings → Pages → Source: Deploy from a branch → Branch: main / root → Save**.
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://<твой-username>.github.io/<имя-репо>/`

Этой ссылкой можно делиться — белого экрана не будет, всё статично.

## Файлы
- `index.html` — разметка
- `styles.css` — стили (тёмная тема, адаптив)
- `app.js` — товары, корзина (localStorage), интерактив
